# PM-5: Market Implied Probability Calibration 分析

## 1. 調查概述
本分析旨在評估 Polymarket 5m BTC 市場價格（Implied Probability）與實際結果（Actual Outcome）之間的校準品質。如果市場價格已經高度校準且無偏，則單純的方向性預測將難以獲利。

## 2. 數據概覽
- **樣本總數**: 500 個已結算的 5m BTC Markets
- **起止時間**: 最近約 2-3 天的市場數據
- **數據來源**: Gamma API (Metadata) + CLOB API (Price History)
- **校準價格點**: 取得市場結算前（跳轉至 0/1 前）的最後有效成交價/中間價作為 Implied Probability。

## 4. 統計指標
- **Overall Brier Score**: 0.2489
- **Baseline Brier Score (50% Guess)**: 0.2500
- **改善幅度**: 0.46% (相較於純隨機預測)

## 5. 可操作性評估
1. **市場效率**: 市場在 40-60% 核心區間表現出極高的效率，Deviation 僅在 0.2% - 1.2% 之間。
2. **獲利空間**: 根據 PM-4 調查，Taker Fee 約佔交易額的固定比例。如果市場偏差（Deviation）小於 Fee 成本，則單純的校準套利不可行。
3. **模型對比**: 我方現有模型（如 CatBoost v1 10m）在 Binance EC 上的 DA 約 56-58%。若要在此市場獲利，必須證明我方模型在 50/50 區間的預測能力顯著優於 55% 的市場定價。

## 6. 結論

### **結論：🔴 Market is Well-Calibrated (Efficient)**

**詳細說明：**
- 5m BTC 市場在最常出現的 40-60% 價格範圍內，市場價格幾乎完美契合實際勝率（誤差 < 1.2%）。
- 這代表該市場不存在明顯的「群眾心理偏差」或「系統性定價錯誤」供簡單套利。
- **Go/No-Go 決策參考**：
    - 如果純粹尋找市場定價錯誤，結論是 **No-Go**。
    - 如果目標是利用我方更高精度的預測模型進行交易，結論是 **Conditional Go**。但必須注意 Taker Fee 將會吃掉大部分利潤，必須優先考慮 **Maker Order** 策略。
