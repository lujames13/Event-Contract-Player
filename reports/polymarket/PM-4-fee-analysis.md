# PM-4: Fee Structure 完整拆解報告

## 1. Fee 公式與參數精確值

對於 Polymarket 的 5m/15m Crypto Markets，Taker 費率由以下公式決定：

**公式:** `fee (in shares/USDC) = N * feeRate * (p * (1 - p))^exponent`

- **N (Quantity):** 交易的股份數量（每股結算價為 $1）。
- **p (Price):** 入場價格（0 到 1 之間的機率）。
- **feeRate:** **0.25**
- **exponent:** **2**

### 特性分析：
- **最大費率點:** 當 p = 0.50 時，`(0.5 * 0.5)^2 = 0.0625`。每股費用為 `0.25 * 0.0625 = $0.015625`。
- **對投資額的比例:** 在 p = 0.50 時，投資額為 每股 $0.50。有效費率 = `0.015625 / 0.5 = 3.125%`。
- **對應 Payout:** 相較於 Binance EC 10m 的 1.80 payout (55.56% breakeven)，Polymarket 在 p=0.50 時的等效 payout 約為 1.94 (34.375% + 50% = 84.375% winrate needed? No, breakeven analysis below).

## 2. Maker vs Taker 差異

- **Maker:** 完全免費 ($0 Fee)，且享有 **Maker Rebate**。
- **Taker:** 支付上述公式計算的費用。
- **Maker Rebate:** 根據交易量分享 Taker 支付費用的約 20%-100%（具體取決於市場類別，5m 市場目前標註為高比例返利）。自 2026 年起，Post-only 訂單全面支援。

## 4. Polygon Gas Fee 估算

- **交易成本:** 由於交易在 Polygon (Layer 2) 上進行，單筆訂單的 Gas 費用極低（通常 < $0.01）。
- **固定成本:** 每次 Approval (USDC) 約 $0.05。
- **結論:** 對於 $50 以上的頭寸，Gas 費用幾乎可以忽略不計。

## 5. 提前平倉 (Selling before settlement)

Polymarket 允許在 5m 結束前賣出持倉：
- **成本:** 賣出同樣視為 Taker 交易（除非掛 Limit Order 並成交）。
- **Double-Fee:** 如果買入和賣出都是 Taker，總 Fee 支出會翻倍。例如在 p=0.5 入場並在 p=0.6 退出，總費率約為 4.5%。
- **建議:** 除非預期利潤遠超 5%，否則不建議頻繁提前 Taker 平倉。

## 結論

**Polymarket 的總成本在所有 Price 範圍內均顯著低於 Binance EC。**

- 在 p=0.5 (最貴點)，Polymarket 的 breakeven winrate 約為 53.5%，優於 Binance 的 55.6%。
- 隨著價格偏離 0.5，Polymarket 的費用迅速下降，而 Binance EC 無論勝率多少，payout 始終固定。
- **如果能成功實作 Maker (Limit Order)，成本將直接降至 0 甚至為負（返利），相較於 Binance EC 具有壓倒性的數學優勢。**
